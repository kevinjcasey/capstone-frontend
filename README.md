# TriviaFluent

![Project Image](https://i.imgur.com/pytDVVc.jpg)

## Description 

Bar trivia game using a third-party API (see resources) which allows users to choose category, difficulty, and number of questions.

## Technologies
- React
- Redux
- Python
- Django
- postgreSQL
- Bulma
- Deployed via Heroku
